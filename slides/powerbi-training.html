Langkah 2

Bersihkan data dengan Power Query.

// Ubah jenis lajur
Table.TransformColumnTypes(NaikTajuk, {
  {"tarikh_terima", type date},
  {"tarikh_selesai", type date},
  {"amaun_kompaun", Currency.Type}
})

// Tambah lajur tempoh hari
Table.AddColumn(t, "tempoh_hari", each
  if [tarikh_selesai] = null then null
  else Duration.Days([tarikh_selesai] - [tarikh_terima]))

Sebelum muat: hidupkan Data Profiling (View → Column quality / distribution / profile) untuk sahkan tiada nilai kosong, ralat, atau pendua tersembunyi.

Power Query (M)Promote headers → ubah jenis → trim → custom column → profil data → Close & Apply

Hari 2

DAX, Kepintaran Masa & Dashboard

Cipta jadual Kalendar & sukatan DAX, tambah kepintaran masa, bina visual lanjutan, dan terbitkan dashboard ke Power BI Service.

Hari 2Kalendar → Sukatan → Visual → Terbit
Langkah 1

Jadual Kalendar dengan DAX.

Kalendar =
ADDCOLUMNS(
    CALENDAR(DATE(2025,1,1), DATE(2026,12,31)),
    "Tahun", YEAR([Date]),
    "Nama Bulan", FORMAT([Date], "mmm"),
    "Tahun-Bulan", FORMAT([Date], "YYYY-MM")
)
Kemudian buat hubungan aduan[tarikh_terima] → Kalendar[Date], dan tanda sebagai Date table.
Modeling > New tableAsas untuk semua kepintaran masa
Konsep penting

Sukatan vs Lajur Terkira.

Calculated Column

Dikira baris demi baris, disimpan dalam jadual. Guna untuk mengkategori setiap rekod (cth: Kumpulan Tempoh).

Measure (Sukatan)

Dikira mengikut konteks penapis visual, tidak disimpan. Guna untuk agregat (cth: Jumlah Aduan, % Selesai).

Petua: jika ia satu nombor ringkasan yang berubah ikut carta → gunakan measure.
Asas DAXNew column vs New measure
Langkah 3

Sukatan asas.

Jumlah Aduan = COUNTROWS(aduan)

Aduan Selesai =
    CALCULATE([Jumlah Aduan], aduan[status] = "Selesai")

% Selesai = DIVIDE([Aduan Selesai], [Jumlah Aduan])
CALCULATE menukar konteks penapis · DIVIDE selamat daripada ralat bahagi-sifar.
New measureCOUNTROWS · CALCULATE · DIVIDE
Langkah 4

Sukatan kewangan (RM).

Jumlah Kompaun (RM) = SUM(aduan[amaun_kompaun])

Bilangan Kompaun =
    CALCULATE([Jumlah Aduan], aduan[tindakan] = "Kompaun")

Purata Kompaun (RM) =
    DIVIDE([Jumlah Kompaun (RM)], [Bilangan Kompaun])
Paparkan Jumlah Kompaun mengikut agensi dalam bar chart; format sebagai mata wang (RM).
Analisis penguatkuasaanSUM + CALCULATE + DIVIDE
Langkah 5

Kepintaran masa & trend.

Aduan Bulan Lepas =
    CALCULATE([Jumlah Aduan], DATEADD('Kalendar'[Date], -1, MONTH))

Perubahan Bulanan % =
    DIVIDE([Jumlah Aduan] - [Aduan Bulan Lepas], [Aduan Bulan Lepas])
Bina line chart: Jumlah Aduan mengikut Kalendar[Tahun-Bulan] — lihat lonjakan musim tengkujuh & musim kering.
Time intelligencePerlu jadual Kalendar + hubungan tarikh
